# TMUX cheatsheet

Bind key= CTRL-B

## List alle snarveier

CTRL-B ?

### Start new

```bash
tmux
```

### Attach

```bash
tmux a
```

### List

```bash
tmux ls
```

### Tabs

```bash
c  create window
w  list windows
n  next window
p  previous window
f  find window
,  name window
&  kill window
```

### Panes

```bash
%  vertical split
"  horizontal split

o  swap panes
q  show pane numbers
x  kill pane
```
